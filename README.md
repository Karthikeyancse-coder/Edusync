<!-- WHAT IS SECTION -->
## <img src="https://media.giphy.com/media/hvRJCLFzcasrR4ia7z/giphy.gif" width="28"/> What is EduSync?

EduSync is a **full-stack college management & communication platform** built for real educational institutions. It replaces dozens of fragmented tools — WhatsApp groups, paper registers, spreadsheets — with a single, beautiful, role-aware application.

Every user gets a **unique EduSync ID**, a verified role, and a personalized dashboard that surfaces exactly what they need.

<!-- GETTING STARTED -->
## ⚡ Getting Started

<details>
<summary><b>📋 Prerequisites</b></summary>

- Node.js `>= 18`
- A [Supabase](https://supabase.com) project *(free tier works perfectly)*

</details>

### Step 1 — Clone

```bash
git clone https://github.com/Karthikeyancse-coder/Edusync.git
cd Edusync/edusync
```

### Step 2 — Install

```bash
npm install
```

### Step 3 — Environment

Create `.env.local` in the root:

```env
NEXT_PUBLIC_SUPABASE_URL=your_supabase_project_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
```

### Step 4 — Database

```bash
node setup_db.js
```

### Step 5 — Run

```bash
npm run dev
```

> 🌐 Open [http://localhost:3000](http://localhost:3000) and click **Demo Login** to explore instantly.

## 🎨 Design Philosophy

<div align="center">

| Token | Value |
|:---:|:---|
| 🌗 **Dark / Light Mode** | System preference aware, toggle available |
