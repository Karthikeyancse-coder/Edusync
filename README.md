## 📌 What is EduSync?

EduSync is a **full-stack college management & communication platform** built for real institutions. Every user gets a **unique EduSync ID**, a verified role, and a personalized dashboard with exactly what they need — nothing more, nothing less.

## ⚡ Getting Started

<details>
<summary><b>Prerequisites</b></summary>
<br/>

- Node.js `>= 18`
- A free [Supabase](https://supabase.com) project

</details>

```bash
# 1. Clone
git clone https://github.com/Karthikeyancse-coder/Edusync.git
cd Edusync/edusync

# 2. Install
npm install

# 3. Environment — create .env.local
NEXT_PUBLIC_SUPABASE_URL=your_project_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_anon_key

# 4. Database setup
node setup_db.js

# 5. Run
npm run dev
```

> 🌐 Open [http://localhost:3000](http://localhost:3000) → click **Demo Login** to explore any role instantly.
